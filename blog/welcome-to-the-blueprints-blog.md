# Welcome to the Blueprints blog

This feed holds **framework-level** narrative: how to read the handbook, how process pieces connect, and pointers for agents that need context beyond a single Markdown file.

Canonical Markdown for the reusable SDLC lives in this repository under `sdlc/`, `pdlc/`, `disciplines/`, and related trees. When that structure is hard to navigate, start here or use the [Forge product blog](https://forgesdlc.com/blog/index.html) for product-facing context.
