# Blueprints blog

Framework-oriented posts that should **version with the Blueprints repo** live here. This directory is the **canonical** Markdown source for the Blueprints framework blog. This directory is the **authoritative** Markdown source.

The [blueprints-website](https://github.com/autowww/blueprints-website) repo maintains a **full copy** at `blog/` (repository root) for static site generation. After editing here, refresh the submodule in that repo and run `generator/sync-blog-from-blueprints.sh`, then commit the mirrored files.

Published site: [blueprints.forgesdlc.com/blog/](https://blueprints.forgesdlc.com/blog/).
