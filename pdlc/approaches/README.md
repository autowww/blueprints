# PDLC approach guides (blueprint)

**Purpose:** Deeper, **product-agnostic** guides for product development approaches than the short summaries in [`PDLC.md`](../PDLC.md). Each file explains the approach on its own terms, links to **authoritative external** material, and states how it **maps** to PDLC phases P1–P6 and (where applicable) SDLC phases A–F.

**Audience:** Teams adopting [`blueprints/pdlc/`](../README.md); project-specific overrides stay in **`docs/product/`** and project-level planning docs.

| Guide | Topics |
|-------|--------|
| [**Dual-Track Agile**](dual-track-agile.md) | Parallel discovery and delivery tracks; product trio; weekly discovery cadence; sync points with SDLC ceremonies. |
| [**Stage-Gate**](stage-gate.md) | Decision gates between phases; go/kill/pivot criteria; 5th-generation agile-integrated gates; mapping to PDLC G1–G5. |
| [**Design Thinking**](design-thinking.md) | Double Diamond; Empathize → Define → Ideate → Prototype → Test; mapping to P1–P2; human-centered design principles. |
| [**Lean Startup**](lean-startup.md) | Build-Measure-Learn; validated learning; minimum viable experiments; pivot/persevere; hypothesis-driven development. |
| [**Product Lifecycle Management**](product-lifecycle-management.md) | Introduction → Growth → Maturity → Decline; investment strategies per stage; harvest/invest/divest; sunset planning. |
| [**Opportunity Solution Trees**](opportunity-solution-trees.md) | Teresa Torres framework; outcomes → opportunities → solutions → experiments; visual discovery structure; continuous discovery. |

**External references:** [`REFERENCE-LINKS.md`](REFERENCE-LINKS.md) — curated URLs with executive summaries and `curl` verification.

**Bridge to SDLC:** [`PDLC-SDLC-BRIDGE.md`](../PDLC-SDLC-BRIDGE.md) — how these approaches relate to delivery methodologies (Scrum, Kanban, XP, phased) in [`blueprints/sdlc/methodologies/`](../../sdlc/methodologies/README.md).
