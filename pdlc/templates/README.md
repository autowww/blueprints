# PDLC templates

Copy-paste starters for product lifecycle artifacts. Copy into your project's `docs/product/` (or equivalent) and fill in.

| Template | PDLC phase | Purpose |
|----------|-----------|---------|
| [`PRODUCT-VISION.template.md`](PRODUCT-VISION.template.md) | P3 Plan & Commit | Product vision, target audience, value proposition, success criteria |
| [`MARKET-ANALYSIS.template.md`](MARKET-ANALYSIS.template.md) | P1 Discover | Market sizing (TAM/SAM/SOM), segmentation, dynamics, regulatory landscape |
| [`COMPETITIVE-ANALYSIS.template.md`](COMPETITIVE-ANALYSIS.template.md) | P1 Discover | Competitor inventory, feature comparison, positioning, differentiation, moats |
| [`WBS.template.md`](WBS.template.md) | P3, SDLC-A | Work breakdown structure: Themes → Epics → Stories → Tasks with Forge ID mapping |
| [`EXPERIMENT-LOG.template.md`](EXPERIMENT-LOG.template.md) | P1–P2, P5 | Track hypotheses, methods, results, and decisions for discovery experiments |
| [`STAGE-GATE-REVIEW.template.md`](STAGE-GATE-REVIEW.template.md) | G1–G5 | Document go/kill/pivot decisions at PDLC stage gates |
| [`PRODUCT-METRICS.template.md`](PRODUCT-METRICS.template.md) | P3–P5 | Define and track outcome metrics (adoption, retention, revenue, satisfaction) |
| [`GO-TO-MARKET.template.md`](GO-TO-MARKET.template.md) | P3–P4 | Launch strategy, positioning, channels, pricing, rollout plan |
| [`SUNSET-PLAN.template.md`](SUNSET-PLAN.template.md) | P6 | Product retirement: timeline, migration, communications, data handling |
