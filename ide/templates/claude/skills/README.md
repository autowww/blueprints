# Claude Code skill templates

Skill definitions copied to `.claude/skills/` by `init-ide-workspace.sh`. Each skill has its own directory with a `SKILL.md` file.

| Template | Skill | Activation |
|----------|-------|------------|
| `spec-driven/SKILL.template.md` | Spec-driven development | Auto (by description) or `/spec-driven` |
| `plan-and-verify/SKILL.template.md` | Plan-first + verification | Auto or `/plan-and-verify` |
