# Cursor command templates

Command files copied to `.cursor/commands/` by `init-ide-workspace.sh`. Invoked in Cursor chat via `/command-name`.

| Template | Command | What it does |
|----------|---------|-------------|
| `plan.template.md` | `/plan` | Create a plan artifact linked to a work-unit ID |
| `pr.template.md` | `/pr` | Commit with work-unit ref, push, create PR |
| `review-spec.template.md` | `/review-spec` | Compare code against spec acceptance criteria |
