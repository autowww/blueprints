# Ceremonies — blueprint package

**Purpose:** Split **methodology-neutral ceremony *intent*** (foundation) from **methodology-specific names and rules** (forks)—same pattern as [**Roles & archetypes**](../roles-archetypes.md) (archetypes = foundation; Scrum/Kanban/… = tweaks).

| Layer | File | Contents |
|-------|------|----------|
| **Foundation** | [`ceremony-foundation.md`](ceremony-foundation.md) | **Intent types** (C1–C6), phase + archetype matrices, **practice suggestions** per intent, principles. |
| **Bridge** | [`methodology-bridge.md`](methodology-bridge.md) | **Crosswalk** intent ↔ Scrum / Kanban / phased / XP **names**; blend tips; how to map your calendar. |
| **Fork — Scrum** | [`scrum.md`](https://forgesdlc.com/methodology-scrum.html) | Scrum **events** mapped to foundation intents; links to [Scrum guide](https://forgesdlc.com/methodology-scrum.html) + Scrum Guide. |
| **Fork — Kanban** | [`kanban.md`](https://forgesdlc.com/methodology-kanban.html) | Kanban **feedback loops** and common rituals mapped to intents; links to [Kanban](https://forgesdlc.com/methodology-kanban.html). |
| **Fork — Phased** | [`phased.md`](phased.md) | **Gates**, phase reviews, formal assurance mapped to intents; links to [Phased delivery](https://forgesdlc.com/methodologies-phased-delivery.html). |
| **Fork — XP** | [`xp.md`](https://forgesdlc.com/methodology-xp.html) | XP **practices** that behave like ceremonies mapped to intents; links to [XP](https://forgesdlc.com/methodology-xp.html). |
| **Fork — SAFe** | [`safe.md`](https://forgesdlc.com/methodology-safe.html) | SAFe **program-level and team-level events** mapped to intents; links to [SAFe](https://forgesdlc.com/methodology-safe.html). |
| **Fork — Lean** | [`lean.md`](https://forgesdlc.com/methodology-lean.html) | Lean **improvement practices** (Kaizen, value-stream mapping, A3) mapped to intents; links to [Lean](https://forgesdlc.com/methodology-lean.html). |
| **Fork — Spiral** | [`spiral.md`](https://forgesdlc.com/methodologies-spiral.html) | Spiral **quadrant events** and anchor-point milestones mapped to intents; links to [Spiral](https://forgesdlc.com/methodologies-spiral.html). |
| **Fork — V-Model** | [`v-model.md`](https://forgesdlc.com/methodologies-v-model.html) | V-Model **design reviews** and **test gates** mapped to intents; links to [V-Model](https://forgesdlc.com/methodologies-v-model.html). |
| **Fork — DevOps** | [`devops.md`](https://forgesdlc.com/methodologies-devops.html) | DevOps **operational ceremonies** (deployment, incident, SLO) mapped to intents; links to [DevOps](https://forgesdlc.com/methodologies-devops.html). |

**Handbook:** [`../../docs/methodologies-ceremonies.html`](../../docs/methodologies-ceremonies.html) — summary + navigation.
