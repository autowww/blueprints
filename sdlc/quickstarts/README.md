# Quickstarts

**Time to first value:** about **one hour** for the path below — concrete commands in your repository, with checks after each step.

This section answers **“What do I do in the first hour?”** It does **not** replace [Adopting Blueprints](../../docs/ADOPTION.md) (which ICP path A / B / C you are on) or the [Roadmap](../../docs/ROADMAP.md) (what maintainers ship next). Use those when you need **role** or **release** context.

| Quickstart | Outcome |
|------------|---------|
| [**First hour in your repo**](first-hour.md) | `blueprints/` present, project `sdlc/` workspace created, Forge and Cursor alignment started — with verification at each step |
