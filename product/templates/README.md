# Templates — product functional docs

Copy these into **`docs/product/`** in your repository and rename (drop `.template` or keep—your convention).

| File | Use for |
|------|---------|
| [`VISION.template.md`](VISION.template.md) | Problem, goals, non-goals, success metrics. |
| [`PERSONAS.template.md`](PERSONAS.template.md) | Segments or lightweight personas. |
| [`CAPABILITY.template.md`](CAPABILITY.template.md) | One capability in a capability map. |
| [`JOURNEY.template.md`](JOURNEY.template.md) | One end-to-end user journey. |
| [`FEATURE-SPEC.template.md`](FEATURE-SPEC.template.md) | Detailed behavior for a feature or domain. |

After copying, remove or replace placeholder text. Link optional IDs to **`docs/requirements/`** when applicable.
