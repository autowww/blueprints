# PM approaches — reference links

Curated external URLs for the PM approach guides in this directory. Each entry includes an **executive summary** explaining why the link is included and what it contributes to understanding the approach in the context of this blueprint.

**Maintenance:** When a URL breaks, find the replacement and update this file. Run `curl -sI <url> | head -1` to verify HTTP 200.

---

## PMI / PMBOK

| Resource | URL | Executive summary |
|----------|-----|-------------------|
| PMI — PMBOK Guide | https://www.pmi.org/pmbok-guide-standards/foundational/pmbok | **Official** PMBOK standard page — principles, performance domains, and purchasing information. The canonical source for PMI-based project management. |
| PMI — Agile Practice Guide | https://www.pmi.org/pmbok-guide-standards/practice-guides/agile | **Official** PMI guidance on Agile delivery within PM governance — co-developed with the Agile Alliance. Bridges the PMI and Agile communities. |
| PMI — Process Groups Practice Guide | https://www.pmi.org/pmbok-guide-standards/practice-guides/process-groups | **Companion** to PMBOK 7th ed for teams wanting the traditional process-oriented structure (Initiating → Planning → Executing → M&C → Closing). |
| PMI — What is Project Management? | https://www.pmi.org/about/what-is-project-management | **Introductory** definition of PM from PMI's perspective — useful for onboarding team members new to PM vocabulary. |
| PMI — Standards Library | https://www.pmi.org/pmbok-guide-standards | **Full catalog** of PMI standards: PMBOK, OPM3, benefits realization, requirements management, scheduling, risk, and more. |
| Wikipedia — PMBOK | https://en.wikipedia.org/wiki/Project_Management_Body_of_Knowledge | **Neutral overview** of PMBOK history, editions, and relationship to PMP certification — good starting point before the official standard. |
| Wikipedia — Project Management Institute | https://en.wikipedia.org/wiki/Project_Management_Institute | **Context** on PMI as an organization: founding, membership, certifications, global influence. |

|----------|-----|-------------------|

## Six Sigma / Lean Six Sigma

| Resource | URL | Executive summary |
|----------|-----|-------------------|
| ASQ — Six Sigma | https://asq.org/quality-resources/six-sigma | **Practitioner-neutral** overview from the American Society for Quality — DMAIC, tools, belt system, history, and relationship to Lean. |
| ASQ — Lean | https://asq.org/quality-resources/lean | **ASQ** overview of Lean principles and tools — the waste-elimination half of Lean Six Sigma. |
| ASQ — DMAIC Process | https://asq.org/quality-resources/dmaic | **Detailed** walkthrough of the DMAIC cycle: Define, Measure, Analyze, Improve, Control — with tool recommendations per phase. |
| Wikipedia — Six Sigma | https://en.wikipedia.org/wiki/Six_Sigma | **Comprehensive** overview including history (Motorola, GE), methodology, tools, criticism, and industry adoption patterns. |
| Wikipedia — Lean Six Sigma | https://en.wikipedia.org/wiki/Lean_Six_Sigma | **Overview** of the combined methodology — how Lean and Six Sigma complement each other. |
| iSixSigma | https://www.isixsigma.com/ | **Community** hub: articles, case studies, tools, calculators, and forums — practical Lean Six Sigma content for practitioners. |
| IASSC — Lean Six Sigma | https://www.iassc.org/ | **Certification** body: International Association for Six Sigma Certification — belt certification standards and body of knowledge. |
